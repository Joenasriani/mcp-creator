Simple step-by-step guide

  1. Give your integration a short name, such as sales-assistant.
  2. Pick a template if it matches your business tool.
  3. Write abilities in normal words: “Add a lead”, “Search invoices”, “Read product list”.
  4. For each ability, paste the real app link the MCP should use.
  5. Generate the package. If something is missing, the app explains it before creating files.

Important: real one-click app login, live chat testing, auto-repair, and cloud deployment require a hosted backend. This Lite file produces the real MCP package only when real connection details are entered.
